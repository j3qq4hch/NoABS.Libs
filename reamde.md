# No Autodesk BullShit reference library 

This repository contains some libraries for EagleCAD with associated to them 3d models. This thing is meant to be used with [NoABS.Eagle3D](https://github.com/j3qq4hch/NoABS.Eagle3D) to generate beautiful 3d models of the PCBs locally and FAST. All the models in this repository were obtained somewhere on the internet, mostly from GrabCAD and some from the components manufacturers directly. They may be not 100% precise, but they get the job done. The main value here is reference folder where a library reference_packages_3d.lbr is. This library contains only the footprints and metadata to properly orient components' 3d models on the footprints. 

The contents of this repository is a subject of additions and extensions. Everyone is invited to participate. 
You can check what this repo has to offer [here](https://github.com/j3qq4hch/NoABS.Eagle3D).
